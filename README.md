Chrome-qwq
======

### 這玩意是做什麼的？<del>瞎扯</del>

當「qwq」加於句尾，通常會使語氣發生微妙的改變。qwq

它可以表達「可愛」、「想哭」或「無奈」，抑或是「我只想賣個萌」，有時甚至也可表達「呵呵」之意。qwq

例文:

- qwq（可單獨使用）
- 我真可愛 qwq
- 這道題不會做啦 qwq
- 小姐姐可愛 qwq
- 羨慕 qwq
- 啊今天好無聊 qwq
- 你是不是腦子瓦特了 qwq

總之，這個小擴展將在你 Twitter 時間線上適當的推文後方插入「qwq」。qwq

真是個沒用的擴展。qwq
